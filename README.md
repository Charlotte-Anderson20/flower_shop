# flower_shop
Flower Shop is a fully responsive online floral shopping website built using PHP, JavaScript, HTML, CSS, and Bootstrap. It includes all core pages of a modern e-commerce site, offering a smooth browsing experience and a clean UI for customers. 

 Features

 User Pages

Home Page — Hero banner, featured flowers, promotional sections

Shop / Products Page — Flower listings with prices & categories

Product Details Page — Detailed product view with images & descriptions

Cart Page — Add, remove, and update items (PHP sessions + JS)

Checkout Page — Simple order summary and customer information form

About Page — Store information, mission, and brand story

Contact Page — Contact form & store details

Login / Register — Basic authentication (if included)

 Admin Pages

Product management (add/edit/delete)

Category management

Tech Stack

Frontend: HTML5, CSS3, Bootstrap, JavaScript

Backend: PHP

Utilities: PHP sessions for cart logic, reusable components (navbar/footer), custom CSS for styling

Responsive Design

Built with Bootstrap + custom CSS to ensure the website works smoothly on:

Desktop

Tablet

Mobile devices

How to Run

Clone the repository

Move the project folder into your local server directory (e.g., htdocs for XAMPP)

Start Apache in XAMPP

Open in browser:

Order viewing
